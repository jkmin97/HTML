복습


  • 웹의 특징
    • 인터넷을 활용하는 응용 서비스 중의 하나
    • 웹 서버와 웹 브라우저로 구성되는 정보 전달 및 공유 서비스
    • 인터넷상에 정보가 얽혀있는 무형의 정보 네트워크

  • CSS의 구조
    • 선택자(Selector) { 속성이름 : 값 }
    • 폰트, 색, 애니메이션 효과까지 다 속성으로 변경
    • 상속 - 부모 태그에 적용한 속성이 자식 태그들한테까지 영향을 미쳐서 덜 귀찮게 손봐도 된다는 특징.

  • HTML의 특징
    • TAG
      • <,>를 이용해서 표현.
      • 내용에 대한 타입을 나타내는 용도.
      • 태그는 열리고 닫히는 태그 구조를 가지고 있으며 이는 한 쌍.
      • 태그 이름은 대소문자를 구분하지 않으나 보통 소문자로 이용.
      • 각 태그는 시작 태그와 종료 태그 쌍이 서로 교차하면 안 됨.

  • CSS 선택자 유형
    • 태그 선택자 - 태그명 사용.
    • 아이디 선택자 # - 필요한 부분에 유일한 아이디를 지정하고 CSS 적용.
    • 클래스 선택자 . - 정의한 클래스 선택자를 태그에 적용.

  • 구조선택자 태그
종류 설명
E:root 최상위 태그 선택
E:nth-child(n) 순서와 일치하는 E 태그 선택(순서에 다른 태그 포함)
E:nth-last-child(n) 뒤에서부터 순서와 일치하는 E 태그 선택(순서에 다른 태그 포함)
E:nth-of-type(n) 순서가 일치하는 E 태그 선택(순서에 E 태그만 포함)
E:nth-last-of-type(n) 뒤에서부터 순서와 일치하는 E 태그 선택(순서에 E 태그만 포함)
E:first-child 맨 첫번째 태그가 E 태그인 경우 선택(순서에 다른 태그 포함)
E:last-child 맨 뒤의 태그가 E 태그인 경우 선택(순서에 다른 태그 포함)
E:first-of-type E 태그 중 첫번째 선택
E:last-of-type E 태그 중 마지막번째 선택
E:only-child 유일한 자식인 E 태그 선택
E:only-of-type 유일한 타입의 E 태그 선택
E:empty 텍스트와 공백을 포함하여 자식요소가 없는 E 태그 선택
